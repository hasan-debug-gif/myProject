# Eventify - Event Management System

## Project Description
Eventify is a web application that allows users to browse, register, and book tickets for events. Organizers can create and manage events, and administrators can view bookings and messages. The project demonstrates a full-stack application using **ReactJS** for the frontend, **Node.js** with **Express** for the backend, and **MySQL** for the database.

Key Features:
- Browse upcoming events with details (date, time, venue, price, capacity)
- Book tickets for events with total price calculation
- User authentication (login/signup)
- Organizer CRUD operations for events
- Admin panel to view bookings and messages
- Responsive design using Bootstrap
